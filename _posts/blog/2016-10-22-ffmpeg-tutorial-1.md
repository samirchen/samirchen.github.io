## 背景

在 Mac OS 上如果要运行教程中的相关代码需要先安装 FFmpeg，建议使用 brew 来安装：

```
// 用 brew 安装 FFmpeg：
brew install ffmpeg
```

或者你可以参考[在 Mac OS 上编译 FFmpeg][5]使用源码编译和安装 FFmpeg。

教程原文地址：[http://dranger.com/ffmpeg/tutorial01.html][3]，本文中的代码做过部分修正。
